Information site about URIs for specific identifiers used in Region Stockholm

Inledning

Inom Region Stockholm används olika typer av identifierare i befintliga IT-system. Det kan vara olika typer av organisationsidentifierare (t.ex. kombikoder eller enhetsidentifierare i journalsystemet TakeCare) eller personidentifierare (t.ex. personnummer eller reservnummer). När system kommunicerar med varandra bör det vara tydligt för systemen vilken typ av identifierare som överförs, därför behövs ett sätt att "identifiera identifierare". Detta kan göras med hjälp av universella resursidentifierare (URI). För att identifiera identifierare som används lokalt inom regionen har ett antal URI:er skapats, se listan nedan.
