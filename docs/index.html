Information site about URIs for specific identifiers used in Region Stockholm

Introduction

Within Region Stockholm, different types of identifiers are used in existing IT systems. These can be different types of organizational identifiers (e.g. kombika-codes or unit identifiers in the medical record system TakeCare) or personal identifiers (e.g. personal number or reserve number). When systems communicate with each other, it should be clear to the systems what type of identifier is being transmitted, therefore a way to "identify identifiers" is needed. This can be done using universal resource identifiers (URIs).

In order to identify identifiers that are locally used within the region, a number of URIs were created, se the list below.

Existing URIs

The following identifiers are locally used within Region Stockholm. Thus, the given URIs have been created.
